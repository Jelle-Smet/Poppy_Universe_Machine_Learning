# Poppy Universe – Machine Learning

This folder contains all Machine Learning code for Poppy Universe.  
It goes from simulated/random data to meaningful insights and predictions for the recommendation engine.

## Structure

- `Data_Prep/` → Notebooks to create simulated datasets
- `Input_Data/` → Raw & Simulated datasets
- `Models/` → ML models --> Notebooks
- `Output_Data/` → Prediction outputs for the engine
